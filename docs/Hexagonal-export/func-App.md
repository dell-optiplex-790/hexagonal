# Hexagonal.js Documentation
This file is for `new hexagonal.App()`. The function returns an `HexagonalApp` object. Example:
```typescript
import hexagonal from 'hexagonal.js';
const app = new hexagonal.App();
console.log(app);
```
Intended output: a `HexagonalApp` object
Type signature:
```typescript
function App(): HexagonalApp
```
